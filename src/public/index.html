API Documentation

Endpoint 1

Returns "Hello World!"

HTTP method: GET

Endpoint: /example

Example:

GET /example HTTP/1.1

Endpoint 2

Returns "found" if id is "1", "not found" otherwise

HTTP method: GET

Endpoint: /example/:id

Parameters:

  • id: The ID to check. Required.

Example:

GET /example/1 HTTP/1.1

Endpoint 3

Completes a prompt using the OpenAI API

HTTP method: POST

Endpoint: /completion

Body parameters:

  • prompt: The prompt to complete. Required.

Example:

POST /completion HTTP/1.1
Content-Type: application/json

{
  "prompt": "Some prompt text."
}

Endpoint 4

Returns a list of available OpenAI models

HTTP method: GET

Endpoint: /models

Example:

GET /models HTTP/1.1
